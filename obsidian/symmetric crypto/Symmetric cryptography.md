Symmetric cryptography uses the same key for encryption and decryption
# [[Block Cipher]]
Block Ciphers are used to create a cipher text from a plaintext by using a set of operations that result in random looking noise. The cipher text can be reversed into the plaintext by doing the cipher in reverse.
# [[Stream Cipher]]
Stream ciphers produce a random looking string of characters that can be used to efficiently encrypt streams of data.
# [[Hash Functions]]
Hash functions are used to create a fingerprint of a message or data.
# [[Message Authentication]]
Message Authentication is used to create an authentic channel which ensures that messages have not been tampered with.
# [[Authenticated Encryption]]
Authenticated Encryption combines message authentication with encryption. Some data, the associated data, is not encrypted but authenticated.
