 Pseudo Random Number Generators (PRNG) produce randomness that is indistinguishable from true randomness.

They generate randomness in a deterministic way from a small seed, similar to how a minecraft world is generated by the seed.

# Operation
PRNG's operate with a state machine where the state feeds the number generator and part of the output feeds back into the state. The state can be fed from an entropy pool.
![[Pasted image 20260605151218.png]]
Every PRNG has these three operations:
- init() - initializes the entropy pool
- refresh(r) - adds randomness (entropy) to the pool
- next(n) - output n random bits
